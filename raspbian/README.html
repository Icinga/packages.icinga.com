Raspbian for Raspberry Pi

Archive Key

wget -O - https://packages.icinga.com/icinga.key | apt-key add -
# or
curl https://packages.icinga.com/icinga.key | apt-key add -

Release Builds

icinga-buster armhf 

deb http://packages.icinga.com/raspbian icinga-buster main
deb-src http://packages.icinga.com/raspbian icinga-buster main

icinga-stretch armhf 

deb http://packages.icinga.com/raspbian icinga-stretch main
deb-src http://packages.icinga.com/raspbian icinga-stretch main

Or if you are a lazy one:

DIST=$(awk -F"[)(]+" '/VERSION=/ {print $2}' /etc/os-release); \
 echo "deb http://packages.icinga.com/raspbian icinga-${DIST} main" > \
 /etc/apt/sources.list.d/icinga.list
 echo "deb-src http://packages.icinga.com/raspbian icinga-${DIST} main" >> \
 /etc/apt/sources.list.d/icinga.list

Snapshot Builds

These are snapshot builds which are not suitable for production use. If you're setting up a production system please use the release packages instead.

icinga-buster-snapshots armhf 

deb http://packages.icinga.com/raspbian icinga-buster-snapshots main
deb-src http://packages.icinga.com/raspbian icinga-buster-snapshots main

icinga-stretch-snapshots armhf 

deb http://packages.icinga.com/raspbian icinga-stretch-snapshots main
deb-src http://packages.icinga.com/raspbian icinga-stretch-snapshots main

Or if you are a lazy one:

DIST=$(awk -F"[)(]+" '/VERSION=/ {print $2}' /etc/os-release); \
