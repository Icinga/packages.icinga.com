Extra Packages for Enterprise Linux

Target platforms:

  • RedHat Enterprise Linux (RHEL)
  • CentOS
  • Oracle Enterprise Linux (OEL)

Requirements

We require some dependencies from the EPEL repository itself, so make sure to enable it.

CentOS 6 / 7

yum install epel-release

Icinga Web 2 will also require PHP via the SCL repository system:

yum install yum install centos-release-scl

RedHat 6 / 7 and compatible

yum install https://dl.fedoraproject.org/pub/epel/epel-release-latest-6.noarch.rpm

# or for RedHat 6
yum install https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm

Icinga Web 2 will also require PHP via the SCL repository system:

subscription-manager repos --enable rhel-7-server-optional-rpms
subscription-manager repos --enable rhel-server-rhscl-7-rpms

# or for RedHat 6
subscription-manager repos --enable rhel-6-server-optional-rpms
subscription-manager repos --enable rhel-server-rhscl-6-rpms

Release Builds

rpm --import https://packages.icinga.com/icinga.key

yum install https://packages.icinga.com/epel/icinga-rpm-release-7-latest.noarch.rpm

# or for RedHat 6
yum install https://packages.icinga.com/epel/icinga-rpm-release-6-latest.noarch.rpm

Snapshot Builds

WARNING: These are auto-built packages, based on current development trees.
Made only for testing purposes, you should not use them in production environments! 

rpm --import https://packages.icinga.com/icinga.key

wget https://packages.icinga.com/epel/ICINGA-snapshot.repo -O /etc/yum.repos.d/ICINGA-snapshot.repo
