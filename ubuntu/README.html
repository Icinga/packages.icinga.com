Ubuntu

Archive Key

wget -O - https://packages.icinga.com/icinga.key | apt-key add -
# or
curl https://packages.icinga.com/icinga.key | apt-key add -

Release Builds

icinga-bionic i386 amd64 

deb http://packages.icinga.com/ubuntu icinga-bionic main
deb-src http://packages.icinga.com/ubuntu icinga-bionic main

icinga-xenial i386 amd64 

deb http://packages.icinga.com/ubuntu icinga-xenial main
deb-src http://packages.icinga.com/ubuntu icinga-xenial main

Or if you are a lazy one:

. /etc/os-release; if [ ! -z ${UBUNTU_CODENAME+x} ]; then DIST="${UBUNTU_CODENAME}"; else DIST="$(lsb_release -c| awk '{print $2}')"; fi; \
 echo "deb http://packages.icinga.com/ubuntu icinga-${DIST} main" > \
 /etc/apt/sources.list.d/${DIST}-icinga.list
 echo "deb-src http://packages.icinga.com/ubuntu icinga-${DIST} main" >> \
 /etc/apt/sources.list.d/${DIST}-icinga.list

Snapshot Builds

These are snapshot builds which are not suitable for production use. If you're setting up a production system please use the release packages instead.

icinga-bionic-snapshots i386 amd64 

deb http://packages.icinga.com/ubuntu icinga-bionic-snapshots main
deb-src http://packages.icinga.com/ubuntu icinga-bionic-snapshots main
